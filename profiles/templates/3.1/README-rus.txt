# Calculate append=skip

В этой директории размещены шаблоны для новых утилит Calculate, разработанных для дистрибутивов Calculate Linux.

Шаблоны - универсальное средство настройки системы и используются на всех этапах, начиная от сборки дистрибутива и заканчивая настройкой рабочего стола пользователя.

Шаблоны представляют собой файлы и директории. Свойства файлов описываются в строке заголовка, свойства директорий описываются в заголовке служебного файла '.calculate_directory', размещённого внутри директории. Файл, который вы читаете - тоже шаблон, но он не используется для настройки системы, т.к. в заголовке своём содержит параметр 'append' значением 'skip', указывающий утилитам пропускать его. Служебная строка заголовка размещается в начале файла и начинается с '#Calculate'.

В настоящее время используются одновременно несколько версий шаблонов, совпадающих по нумерации с версиями утилит Calculate: 2.0, 2.2, 3.0 и 3.1. Шаблоны 2.0 хранятся в самих пакетах утилит и используются для настройки сервера. Шаблоны 2.2 используются при сборке системы. Шаблоны 3.0 используются установщиком Calculate Linux 12. Шаблоны 3.1 на данный момент являются текущей актуальной версией, поддержка которых со временем должна появиться во всех версиях утилит.

Вы можете использовать свои шаблоны по образу и подобию этих. Ваши шаблоны будут иметь преоритетное значение. Стандартные пути для хранения пользовательских шаблонов: /var/calculate/templates или /var/calculate/remote/templates. Существуют также Clt-файлы шаблонов, располагаемые непосредственно в директории /etc.

Подробнее можно почитать здесь: http://www.calculate-linux.ru/main/ru/calculate_utilities_templates
